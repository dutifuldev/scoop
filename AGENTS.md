# AGENTS.md

- You MUST NOT insert coding agent specific branding, like `[codex]`, in code, PRs or issues created on GitHub.
- For git commits and PR titles that act as the effective merge commit title, use Conventional Commits format: `<type>[optional scope]: <description>`.
- Although `e2b-agents` is inspired by `spritz`, you MUST NOT refer to `spritz` or `textcortex` in `e2b-agents` code, docs, PRs, issues, commits, comments, or other repository artifacts.
- Do not commit, document, or otherwise publish personal details from the local Scoop setup. Private local hostnames, usernames, absolute home-directory paths, private network names, machine names, tokens, private local service URLs, and other person- or machine-specific values must stay in local environment variables, ignored files, or private notes.
